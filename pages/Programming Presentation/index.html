  1. Firstly, you need to decide what kind of website you want to create.
  2. Then, choose a topic, also you can create the team.
  3. After that, buy or make the hosting of your site and start working with design.
  4. When you design will be finished, start writing body of webpages using HTML/CSS and sometimes JavaScript.
  5. After Frontend start develop a Backend part of your website using only ONE language. I will use "Python" on this website later.
  6. And finally, test your site, host it, publish and add donate page.
